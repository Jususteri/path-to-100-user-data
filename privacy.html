Tietosuojaseloste

Viimeksi päivitetty: 3.9.2025

Path to 100 sitoutuu suojaamaan käyttäjiensä yksityisyyttä. Tämä tietosuojaseloste kuvaa, miten peli käsittelee käyttäjätietoja.

1. Mitä tietoja kerätään

Peli ei kerää tietoja ilman käyttäjän lupaa. Pelaajalla on mahdollisuus tallentaa pelin päättymisen jälkeen seuraavia tietoja: peliaika, pisteet, peruutukset ja nimimerkki. Käyttäjälle annetaan poistokoodi näiden tietojen poistamista varten. Tämä koskee niin Satakerhoa, Kellokerhoa, Jahtikerhoa kuin Sumakerhoakin.

Sovellus käyttää paikallista muistia tietojen tallentamiseen ja hallintaan, mikä vähentää tietokantakyselyiden määrää ja parantaa sovelluksen suorituskykyä. Tämä tieto ei ole jaettavissa sovelluksen ulkopuolelle ilman käyttäjän nimenomaista suostumusta.

2. Tietojen säilytysaika

Tallennettuja tietoja säilytetään, kunnes ne eivät enää ole tarpeen pelin toiminnan kannalta. Tietokantaan voidaan tallentaa enintään 1000 pelaajan tulokset. Tietoja voidaan poistaa seuraavissa tilanteissa:

  • Jos käyttäjä poistaa tietonsa käyttämällä uniikkia poistokoodiaan. Lue lisää
  • Jos tietokanta saavuttaa maksimikapasiteettinsa, vanhimpia tai huonompia tuloksia voidaan poistaa vapauttamaan tilaa.
  • Jos nimimerkki sisältää loukkaavaa, sopimatonta tai hyvän maun vastaista sisältöä, nimimerkki ja siihen liittyvät tulokset voidaan poistaa.
Pelaajan tulos voi säilyä tietokannassa niin kauan, kun se kuuluu pelin 100 parhaan tuloksen joukkoon, mutta jos tulos jää tämän ulkopuolelle, se voidaan poistaa tietokannan hallinnan yhteydessä.

Privacy Policy

Last updated: 03/09/2025

Path to 100 is committed to protecting user privacy. This privacy policy explains how the game handles user data.

1. Data Collected

The game does not collect any data without user consent. After completing the game, the player may save the following data: game time, score, undos, and nickname. A unique deletion code is provided for removing this data. This applies to the Hundred Club, Clock Club, Hunt Club, and Logjam Club.

The app utilizes local storage to manage and store data, which reduces database queries and improves app performance. This information is not shared outside the app without explicit user consent.

2. Data Retention

Saved data is retained as long as necessary for the functionality of the game. The database can store up to 1000 player results. Data may be removed in the following situations:

  • If the user deletes their data using their unique deletion code. Read more
  • If the database reaches its maximum capacity, older or poorer results may be removed to free up space.
  • If a nickname contains offensive, inappropriate, or objectionable content, the nickname and related results may be removed.
A player's result can remain in the database as long as it is among the top 100 results in the game. If it falls outside this range, it may be removed during database maintenance.

Integritetspolicy

Senast uppdaterad: 2025-09-03

Path to 100 är engagerad i att skydda användarnas integritet. Denna integritetspolicy förklarar hur spelet hanterar användardata.

1. Insamlad data

Spelet samlar inte in någon data utan användarens samtycke. Efter att ha klarat spelet kan användaren spara följande data: speltid eller poäng, ångringar och smeknamn. En unik raderingskod tillhandahålls för att ta bort data. Detta gäller Hundraklubben, Klockklubben, Jaktklubben samt Stockklubben.

Appen använder lokal lagring för att hantera och lagra data, vilket minskar antalet databasfrågor och förbättrar appens prestanda. Denna information delas inte utanför appen utan uttryckligt användarsamtycke.

2. Lagringstid för data

Sparad data lagras så länge det är nödvändigt för spelets funktion. Databasen kan lagra upp till 1000 spelares resultat. Data kan tas bort i följande situationer:

  • Om användaren tar bort sina data med sin unika raderingskod. Läs mer
  • Om databasen når sin maximala kapacitet kan äldre eller sämre resultat tas bort för att frigöra utrymme.
  • Om ett smeknamn innehåller kränkande, olämpligt eller osmakligt innehåll kan smeknamnet och relaterade resultat tas bort.
En spelares resultat kan finnas kvar i databasen så länge det är bland de 100 bästa resultaten i spelet. Om det hamnar utanför detta intervall kan det tas bort under databasunderhåll.

Privaatsuspoliitika

Viimati uuendatud: 03.09.2025

Path to 100 kaitseb kasutaja privaatsust. See privaatsuspoliitika selgitab, kuidas mäng kasutajaandmeid käsitleb.

1. Kogutavad andmed

Mäng ei kogu andmeid ilma kasutaja loata. Pärast mängu lõppu saab mängija salvestada järgmised andmed: aeg või punktid, tühistamised ja kasutajanimi. Andmete kustutamiseks antakse unikaalne kood. See kehtib Sajaklubi, Kellaklubi, Jahiklubi ning Palgiklubi kohta.

Rakendus kasutab andmete haldamiseks kohalikku salvestusruumi, mis vähendab andmebaasipäringuid ja parandab jõudlust. Andmeid ei jagata väljaspool rakendust ilma kasutaja selgesõnalise nõusolekuta.

2. Andmete säilitamine

Salvestatud andmeid säilitatakse seni, kuni need on vajalikud mängu toimimiseks. Andmebaasi saab salvestada kuni 1000 mängija tulemust. Andmeid võidakse kustutada järgmistel juhtudel:

  • Kui kasutaja kustutab oma andmed, kasutades unikaalset kustutamiskoodi. Loe lisa
  • Kui andmebaas saavutab maksimaalse mahutavuse, võidakse vanemad või nõrgemad tulemused eemaldada.
  • Kui kasutajanimi sisaldab solvavat, sobimatut või ebasobivat sisu, võidakse see ja seotud tulemused eemaldada.
Mängija tulemus võib andmebaasi jääda seni, kuni see kuulub mängu 100 parima tulemuse hulka. Kui see jääb väljapoole seda vahemikku, võidakse see kustutada andmebaasi hoolduse käigus.
